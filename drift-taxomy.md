# Operational Drift Taxonomy

## 1. Silent Failure Drift
System stops progressing without explicit failure.

## 2. Semantic Drift
Execution continues while operational meaning collapses.

## 3. Governance Drift
Displayed controls diverge from actual enforcement behavior.

## 4. Observability Drift
Monitoring and visibility layers diverge from operational reality.

## 5. Recursive Drift
Self-referential execution patterns create operational deadlocks.

## 6. State Desynchronization
Different system layers report contradictory realities.

## 7. Recovery Drift
Recovery mechanisms themselves become sources of instability.

## 8. Metric Drift
Success metrics diverge from actual operational outcomes.
